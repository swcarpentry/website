A double handful of people associated with Software Carpentry gave talks at PyCon 2014 in Montreal two weeks ago. Thanks to Sheila Miguez and Will Kahn-Greene (who run the excellent PyVideo site), you can now view their presentations:

  • Julia Evans: Diving into Open Data with IPython Notebook & Pandas
  • Olivier Grisel and Jake Vanderplas: Diving deeper into Machine Learning with Scikit-learn and Exploring Machine Learning with Scikit-learn
  • Jessica Hamrick: Games for Science: Creating interactive psychology experiments
  • Preston Holmes: Blending art, technology, and light, Python for interactive and real time
  • Jessica McKellar: Keynote, Building and breaking a Python sandbox, Contribute with me! Getting started with open source development, and Hands-on intro to Python for beginning programmers
  • Fernando Pérez: Keynote and IPython in depth: high productivity interactive and parallel python
  • Greg Ward: Unit Testing Makes Your Code Better
  • Greg Wilson: Software Carpentry: Lessons Learned
