This post originally appeared on the Software Carpentry website.

A few weeks ago we announced that we would help NumFOCUS apply to Google Summer of Code (GSoC) as a mentoring organization. Yesterday the list of mentoring organization was announced and NumFOCUS had been selected.

I want to thank very much (in alphabetical order) to Andy Terrel, Ethan White, Matt Sottile, and W. Trevor King for volunteering to be mentors and making this possible. We have a list of ideas here but students can apply with their own ideas as well—in this case, follow these instructions so we can help polish ideas. Applications on any project supported by NumFOCUS are welcome.

Important dates:

16 March: 19:00 UTC Student application period opens.
27 March: 19:00 UTC Student application deadline.
