This post originally appeared on the Software Carpentry website.

Damian Rouson is teaching a class in Berkeley March 26-28, 2012, and again April 10-12, on object-oriented programming in Fortran 2003. The March class is full, but there are still seats available for April. Fortran is still the workhorse of scientific computing, and recent dialects like 2003 have a lot of powerful features—it's well worth checking out.
